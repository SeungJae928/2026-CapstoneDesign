# 검색 흐름 다음 작업

## 결정 필요

검색어가 지역이 아닐 때 지역 탭을 비워둘지, 현재 위치 기반 랭킹 shortcut을 보여줄지 결정해야 한다.

현재 구현 기준은 비워두는 것이다.
현재 위치 기반 랭킹을 보여주려면 검색 API 계약을 확장해야 한다.

## 후보 API 확장

- `GET /search?query=돈까스&currentRegionName=용인시 처인구`
- 또는 클라이언트가 `GET /rankings/restaurants?regionName=용인시 처인구`를 별도 호출

## 주의

- 지역 탭은 검색어와 무관한 추천 탭이 아니다.
- 검색과 리스트 추가 플로우는 분리해야 한다.
- 사진 탭은 현재 백엔드 검색 응답에 없다.
